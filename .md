# Product Requirements Document (PRD)

# Project

Candy Crush-Style Match-3 Game for Jacks Club

# Version

1.0

# Status

Draft

# Owner

Jacks Club

# Platform

Web (Desktop + Mobile)

# Technology

* SvelteKit
* TypeScript
* HTML5 Canvas (optional)
* GSAP (animations)
* Howler.js (audio)

---

# 1. Overview

Develop a highly polished Match-3 puzzle game inspired by Candy Crush.

The game should prioritize:

* Smooth gameplay
* Fast feedback
* High retention
* Mobile-first experience
* Reward-driven progression

The game must run inside modern browsers without requiring installation.

# 5. Core Gameplay Loop

1. Player starts level
2. Swaps candies
3. Creates matches
4. Earns points
5. Triggers combos
6. Completes objectives
7. Receives rewards
8. Unlocks next level
9. Repeats

---

## Tile Types

### Standard Candies

* Yellow

* Orange

### Special Candies

* Striped Candy
* Wrapped Candy
* Color Bomb

---

# 7. Match Rules

## Valid Matches

### Match 3

* Remove matched candies

### Match 4

* Create Striped Candy

### Match 5

* Create Color Bomb

### L-Shape

* Create Wrapped Candy

### T-Shape

* Create Wrapped Candy

# 8. Level Types

## Collection Level

## Combo Level

Reach goal before timer expires.
